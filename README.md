# TalentMatcher

TalentMatcher compares a resume against a job description and returns an AI-generated compatibility report.

## What it does

- accepts resume text and job description text
- sends both texts to `POST /api/analyze`
- uses Gemini to score relevance, identify strengths and gaps, and suggest improvements
- returns a validated JSON report and resume recommendation data

## How it works

- `src/` is the Next.js frontend
- `api/src/` is the Express backend (proxied via Next.js rewrites)
- the backend registers TSOA routes, calls Gemini, and validates output
- API docs are generated by TSOA and available only in development via `npm run dev:docs`

## Quick start

```bash
git clone https://github.com/zehramagalhaes/talentmatcher.git
cd talentmatcher
npm install
cp .env.example .env.local
npm run dev
```

### Local API docs

```bash
npm run dev:docs
```

> The backend must be running for the API and docs to work.

## Configuration

- copy `.env.example` to `.env.local`
- set `GEMINI_API_KEY`
- optionally set `GEMINI_API_KEY_ALT`
- set `NEXT_PUBLIC_API_BASE_URL` (optional, defaults to `/api`)
- set `API_BASE_URL` (optional, for Next.js proxy to backend)

## Development commands

```bash
npm run dev
npm run dev:docs
npm run dev:api
npm run dev:app
npm run build
npm start
```
